## Main Window

The main window features a modern tabbed interface with three main sections:

### 1. Upload Tab

<p align="center">
  <img src="https://via.placeholder.com/600x400/2d3436/00b894?text=Upload+Tab" alt="Upload Tab Screenshot" width="600"/>
</p>

- Select files or folders using the file browser
- View selected files with sizes
- Configure remote path
- Real-time upload progress with speed metrics

### 2. Download Tab

<p align="center">
  <img src="https://via.placeholder.com/600x400/2d3436/fdcb6e?text=Download+Tab" alt="Download Tab Screenshot" width="600"/>
</p>

- Browse your pCloud files
- Select multiple files for download
- Choose local download location
- Parallel downloads with progress tracking

### 3. Browse Tab

<p align="center">
  <img src="https://via.placeholder.com/600x400/2d3436/a29bfe?text=Browse+Tab" alt="Browse Tab Screenshot" width="600"/>
</p>

- Navigate your pCloud folder structure
- View file/folder hierarchy
- Quick navigation with up/refresh buttons
- Clean, organized file listing

## Authentication Dialog

<p align="center">
  <img src="https://via.placeholder.com/400x300/2d3436/74b9ff?text=Login+Dialog" alt="Login Dialog Screenshot" width="400"/>
</p>

- Secure login with email and password
- Region selection (US/EU)
- Clean, centered modal design
- Auto-saves authentication token

## Progress Dialog

<p align="center">
  <img src="https://via.placeholder.com/500x350/2d3436/ff7675?text=Progress+Dialog" alt="Progress Dialog Screenshot" width="500"/>
</p>

- Real-time transfer progress bar
- **Per-file progress tracking** with current filename display
- Live speed metrics (MB/s)
- Files completed counter (e.g., "5/20 files")
- Percentage complete indicator
- Cancel button for stopping transfers

## Settings Panel

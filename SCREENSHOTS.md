# pCloud Fast Transfer - GUI Screenshots

## Main Window

The main window features a modern tabbed interface with three main sections:

### 1. Upload Tab
- Select files or folders using the file browser
- View selected files with sizes
- Configure remote path
- Real-time upload progress with speed metrics

### 2. Download Tab
- Browse your pCloud files
- Select multiple files for download
- Choose local download location
- Parallel downloads with progress tracking

### 3. Browse Tab
- Navigate your pCloud folder structure
- View file/folder hierarchy
- Quick navigation with up/refresh buttons
- Clean, organized file listing

## Authentication Dialog

- Secure login with email and password
- Region selection (US/EU)
- Clean, centered modal design
- Auto-saves authentication token

## Progress Dialog

- Real-time transfer progress bar
- Current file indicator
- Live speed metrics (MB/s)
- Files completed counter
- Estimated time remaining (ETA)
- Cancel button for stopping transfers

## Settings Panel

- Parallel workers slider (1-16 workers)
- Appearance mode selector (Dark/Light/System)
- Real-time preview of settings
- Easy save functionality

## Theme Support

The GUI supports multiple appearance modes:
- **Dark Mode** (default): Easy on the eyes, modern look
- **Light Mode**: Clean, bright interface
- **System Mode**: Automatically matches OS preference
